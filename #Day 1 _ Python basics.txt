#1 Data Types


Attribute of data which tell the compiler or an interpreter how the programmer intends to use the data.


Examples : int, float, bool, str, list, tuple, set, dict


Text type : str


Numeric type : int, float, complex


Sequence type : list, tuple, range


Mapping type : dict


Boolean type : bool


Binary type : bytes


None type : NoneType


Above data types are fundamental data types.
